# Getting Started

This guide will help you integrate Annota into your application in just a few minutes.

## Installation

Install Annota and its peer dependencies:

```bash
npm install annota openseadragon react react-dom
```

Or with pnpm:

```bash
pnpm add annota openseadragon react react-dom
```

## Quick Start with React

Here's a minimal example to get you started:

```tsx
import { useState } from 'react';
import OpenSeadragon from 'openseadragon';
import { AnnotaProvider, Annotator, AnnotationEditor } from 'annota';

function App() {
  const [viewer, setViewer] = useState<OpenSeadragon.Viewer>();

  return (
    <AnnotaProvider>
      <Annotator
        viewer={viewer}
        onViewerReady={setViewer}
        tileSources="/path/to/image.dzi"
      />
      <AnnotationEditor viewer={viewer} />
    </AnnotaProvider>
  );
}
```

That's it! You now have a working annotation viewer with:
- Point, Rectangle, and Polygon drawing tools
- Drag-and-drop editing of shapes
- Selection management

## Core Concepts

### AnnotaProvider

The `AnnotaProvider` component wraps your application and provides the annotation context:

```tsx
<AnnotaProvider>
  {/* Your annotation components */}
</AnnotaProvider>
```

### Annotator

The `Annotator` component initializes OpenSeadragon and manages the annotation layer:

```tsx
<Annotator
  viewer={viewer}
  onViewerReady={setViewer}
  tileSources="/image.dzi"
  options={{
    showNavigationControl: false,
  }}
/>
```

### AnnotationEditor

The `AnnotationEditor` enables visual editing of annotations:

```tsx
<AnnotationEditor viewer={viewer} />
```

## Using Annotation Tools

Annota provides ready-to-use annotation tools:

```tsx
import { PointTool, RectangleTool, PolygonTool } from 'annota';

function ToolBar({ viewer }: { viewer: OpenSeadragon.Viewer | undefined }) {
  const [currentTool, setCurrentTool] = useState<any>(null);

  return (
    <div>
      <button onClick={() => setCurrentTool(new PointTool())}>
        Point
      </button>
      <button onClick={() => setCurrentTool(new RectangleTool())}>
        Rectangle
      </button>
      <button onClick={() => setCurrentTool(new PolygonTool())}>
        Polygon
      </button>

      {currentTool && viewer && (
        <ToolController viewer={viewer} tool={currentTool} />
      )}
    </div>
  );
}
```

## Working with Annotations

Use hooks to access and manage annotations:

```tsx
import { useAnnotations, useSelection } from 'annota';

function AnnotationList() {
  const annotations = useAnnotations();
  const selection = useSelection();

  return (
    <ul>
      {annotations.map(ann => (
        <li key={ann.id}>
          {ann.shape.type} - {ann.id}
          {selection.find(s => s.id === ann.id) && ' (selected)'}
        </li>
      ))}
    </ul>
  );
}
```
