# Changelog

All notable changes to Annota will be documented here.

## Version History

Each release is documented with detailed changes and installation instructions.

Browse the release notes by version:

- [v0.3.2](/docs/changelog/v0_3_2) - Latest Release (October 26, 2025)
  - Smart click behavior for vertex editing
  - Polygon vertex editing mode with conditional vertex display
  - Refactored hooks into individual files for better maintainability

- [v0.3.1](/docs/changelog/v0_3_1) - Context Menu Update (October 26, 2025)
  - Context menu improvements and style function precedence fixes
  - New useContextMenuBinding hook

- [v0.3.0](/docs/changelog/v0_3_0) - Context Menu Release (October 26, 2025)
  - Context menu system
  - Mask polarity support

- [v0.2.1](/docs/changelog/v0_2_1) - First Major Release (October 26, 2025)
  - Complete annotation framework with undo/redo system
  - High-performance rendering with 10,000+ annotations
  - Polygon vertex editing capabilities
  - Multi-layer organization and management
