# useLayer

Get a specific layer by ID.

## Signature

```typescript
function useLayer(id: string): Layer | undefined
```

## Parameters

- `id` - The layer ID

## Returns

The layer, or `undefined` if not found.

## Example

```tsx
import { useLayer } from 'annota';

function LayerOpacityControl({ layerId }: { layerId: string }) {
  const layer = useLayer(layerId);

  if (!layer) return null;

  return <div>Opacity: {layer.opacity}</div>;
}
```
